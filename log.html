Log File Viewer

The log file viewer opens if you double-click on a log file (file with the extension .log) or open the log file via the context menu.

The log file viewer displays the content of the log file.

The following buttons – in the upper-left corner of the opened viewer – support navigation through errors/warnings in the content:
